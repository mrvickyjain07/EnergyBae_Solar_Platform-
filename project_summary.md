# Project Description

I built the **EnergyBae Solar Platform**, a full-stack web application that uses an AI-powered OCR scanner to analyze MSEDCL electricity bills and automatically generate precise solar PV sizing recommendations, financial savings, and payback projections. The platform was developed using **React, Vite, and Tailwind CSS** for the frontend, alongside a robust backend built with **Node.js, Express, and Python** to handle complex logic. I integrated cutting-edge AI tools, including **Google Gemini 3.5 Flash Multimodal Vision** for accurate data extraction and **ChatGPT** for advanced language processing and user interactions. For future improvements, I plan to integrate a database to save calculation histories, implement user authentication, and add highly advanced features like predictive energy load forecasting and multi-state utility bill support.
